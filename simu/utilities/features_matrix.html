Features matrix simulation¶

mlpp.simulation.features_normal_cov_uniform(n_samples: int = 200, n_features: int = 30)[source]

An example of features obtained as samples of a centered Gaussian vector with a specific covariance matrix given by O.5 * (U + U.T), where U is uniform on [0, 1] and diagonal filled by ones.

Parameters:

n_samples : int, default=200

Number of samples

n_features : int, default=30

Number of features
Returns:

output : numpy.ndarray, shape=(n_samples, n_features)

n_samples realization of a Gaussian vector with the described covariance
